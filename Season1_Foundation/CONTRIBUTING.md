# 🤝 Contributing to LLM Masterclass

이 레포지토리에 오신 것을 환영합니다! 이 프로젝트는 언어 모델(LLM)의 기초부터 심화까지 누구나 쉽게, 하지만 깊이 있게 공부할 수 있도록 돕는 오픈소스 교재입니다.

## 🚀 기여 방법 (How to Contribute)

어떤 형태의 기여든 대환영입니다!
1. **오타 및 문맥 수정:** 마크다운 이론 파일(`01_theory.md`)의 오타나 어색한 문장을 고쳐주세요.
2. **코드 개선:** 파이토치 알고리즘(`02_algorithm_from_scratch.ipynb`)에서 더 효율적인 텐서 연산법이 있다면 PR을 날려주세요.
3. **이슈 제기:** 코드를 돌리다가 에러가 나거나, 수식에 수학적 오류가 있다면 망설이지 말고 Issue를 생성해 주세요.

## 🛠️ Pull Request (PR) 프로세스

1. 이 레포지토리를 **Fork** 합니다.
2. 로컬에 Clone 한 뒤, 새로운 브랜치를 생성합니다. (`git checkout -b feature/amazing-feature`)
3. 변경 사항을 커밋합니다. (`git commit -m 'feat: Add some amazing feature'`)
4. 브랜치를 Push 합니다. (`git push origin feature/amazing-feature`)
5. GitHub에서 **Pull Request**를 엽니다.

여러분의 작은 기여가 전 세계 AI 초심자들에게 큰 빛이 됩니다! 감사합니다.
